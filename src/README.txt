[Description]
This project has 3 java files: Passwords, Table and PasswordGenerator. Passwords contains the main method and just parses the arguments. Table reads the reference text, generates the tables and stores these in tables.tbl. PasswordGenerator has its own local copies of the tables and uses these to generate passwords.
The program can be run using the command:
java Passwords <reference text name> <no of passwords> <length of each password>

[Finish]
The project is complete

[Source of Reference File]
The reference file I used was A Tale of Two Cities. I got it from Project Gutenberg: http://www.gutenberg.org/ebooks/98
The file is 774 kB and has about 138879 words.
